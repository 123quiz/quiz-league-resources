Quiz Setting

Welcome

This is a tool to compile quizzes for use in league matches. It generates a password protected file that the person reading the quiz unlocks, reads from, and uses to record the score. The scoresheet can be quickly shared.

  • Runs in a web browser - no special software required.
  • A template generator is available: TeamQuizTemplate.html
  • Import the quiz questions, answers and data from a spreadsheet in one go.
  • Underlining of the required part(s) of the answer, other text formatting and editing takes place in the browser.
  • The quiz is locked and downloaded as an HTML file for distribution.
  • All sensitive information, including Question and Answer text is encrypted using the Advanced Encryption Standard.
  • Unlike with paper scoresheets, full information on 1s and spares is recorded.
  • A clock displaying seconds can be used for timing.
  • Handicap team start scores can be used, as can individual player start scores.
  • The reader sees the quiz only after recording which team is going first in the match.
  • Each question is prefixed with the player's name.
  • Questions can't be asked out of turn as future questions are hidden from view.
  • An asterisk is shown when a team gets a roundhouse - two points for every player's question in a round.
  • Share the result easily when the quiz is finished by email link or QR code.
  • Works with quiz sets written for non-standard numbers of rounds and players.
  • Coded using open-source libraries. All code can be verified.
  • No back doors or phoning home - network activity can be monitored and the quiz can be locked offline.

Formatting is not required at this stage. Additional columns (Sort, Index, Num, Level, Pair, Subject) should be included if available, so this data can be used later. The following column names must exist, with complete correctly sorted content:

  • Round 1,2,3,4,5,6,7,8 for rounds, S for spares
  • Player 1A,1B,2A,2B,3A,3B,4A,4B in rounds, 1,2,3,4 for spares
  • Question Text
  • Answer Text

The Lock and Download button generates two files (based on the current date and time, to avoid duplicates):
1. TeamQuiz_YYYY-MM-DD_hh-mm-ss_EDIT_THIS_FILE_NAME.html
2. Password_YYYY-MM-DD_hh-mm-ss_SHARE_WITH_CARE.txt
The file name of the first file should be edited to make it clear what this quiz is for. Something of the form: TeamQuiz_2018-2019_League_Week_01_Match.html might be a good name. The extension .html should not be edited.
The contents of the password file should be shared with care as anyone with the quiz file and full password can view the quiz. The suggested method is for the first half of the password to be sent to the home captain and second half of the password to be sent to the away captain.
